# Meta Ad Performance Analysis Dashboard

## 📌 Project Overview
Power BI dashboard analyzing Meta (Facebook & Instagram) ad performance.

## 🔧 Tools Used
- Power BI
- Excel

## 📊 Key Metrics
- Impressions: 216K
- Clicks: 25.4K
- CTR: 11.76%
- Conversion Rate: 5.21%

## 🔍 Key Insights
- High engagement but low purchase rate (0.61%)
- Females (43%) are most engaged
- Age group 20–30 performs best
- Video ads perform best
